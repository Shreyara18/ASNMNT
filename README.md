## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Shreyara18/ASNMNT.git
```

Navigate to the repository

```bash
cd ASNMNT
```

Open the frontend assignments directly in your browser or use **VS Code Live Server**.

For Java assignments:

```bash
cd JAVA
javac main.java
java main
```

For JavaScript assignments:

```bash
node reverse.js
node sorting.js
```
